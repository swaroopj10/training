Link examples

Table of Contents

The whole table of contents consists of links, where each one is anchored by the text in a list item.

Links to Other Websites

Links can direct you anywhere. In this sentence, you can choose which search engine you would like to be taken to, either Google or DuckDuckGo. These two links require a little extra "magic" to open in a new tab. We don't always have to open pages in a new tab, but in this case we do because of how we have implemented the table of contents. We will learn more about this shortly.

Links within the Page

Links can even take you somewhere else in the same document. There are other examples of this on the breaks page.

Image as Link

The photo is a link. If you click on it, you will be taken to the page about images.
